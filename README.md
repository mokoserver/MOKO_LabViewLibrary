# MOKO_LabViewLibrary

Общая библиотека вспомогательных VI для использования в различных проектах на LabVIEW в экосистеме MOKO.

## Описание

Этот проект не является самостоятельным приложением, а представляет собой набор переиспользуемых подпрограмм (SubVI), которые решают общие задачи. Судя по названиям файлов, библиотека включает функции для:

- **Работы со строками:** Поиск чисел с плавающей точкой (`Find_float.vi`).
- **Взаимодействия с MOKO SE:** Обработка команд для различных модулей (HEX, JSON, STM).

Наличие этой библиотеки позволяет избежать дублирования кода и стандартизировать выполнение типовых операций в разных проектах.

## Структура проекта

- **Find_float.vi**: VI для поиска чисел в строке.
- **MHEX_command.txt**, **MJSN_command.txt**, **MSTM_command.txt**: Текстовые файлы, возможно, содержащие описания форматов команд.
- **PicVi/**: Папка с изображениями для использования в VI.

## Ключевые технологии

- LabVIEW
